soundVis.js - A Sound Visualizer

example api

Getting Started

To use this library, you need to include 'soundVis.js' and 'style.css' in your HTML
For example, you can add the folowing lines of code to your html:
< script src="./soundVis.js" > < /script >
< link rel="stylesheet" href="./style.css" >

Variables

all the global variables

Property Type Description
AudioInfo.audioPath String The path that stores the audio
AudioInfo.div Object The whole sound Visualizer object that you can modify or get its properties
AudioInfo.controlDiv Object The control object that you can modify or get its properties
AudioInfo.canvasDiv Object The canvas object that you can modify or get its properties
AudioInfo.headderDiv Object The headder object of the sound visualizer that you can modify or get its properties
AudioInfo.infoDiv Object The pop-up info object of the sound visualizer that you can modify or get its properties
AudioInfo.size Int An integer that represents different choice of the size of the visualizer
(0 = small, 1 = medium, 2 = large)
AudioInfo.theme String An string that represents different choice of the theme of the visualizer
(yellow, blue, pink)
AudioInfo.audioCtx Object The audio content object of the audio we are visualizing
AudioInfo.requestId Int The request id to stop drawing to the canvas object of the visualizer
AudioInfo.id Int The id of the visualizer
AudioInfo.background_color String The background color of the canvas object
AudioInfo.colors List of String The list of colors of the strips that the canvas object display as the audio plays
AudioInfo.curr_color int The index of the current color that the canvas object display as the audio plays

Functions

all functions of the API

const sg = new ShapeGenerator();
Create a new shape generator element

id = sg.makeShape("audiopath.mp3", "div_id", "Audio_Name");
make a small visualizer with yellow theme that gets the audio from audio path audiopath.mp3 in a html block with id div_id and audio name Audio_Name. This method will return the id(that is stored in data but not HTML) of the object created

sg.deleteShape(id);
delete the shape with the specific id

sg.changeTheme(id, "blue");
change the theme of the visualizer with the specific id. The theme can be ("blue", "yellow", or "pink")

sg.changeSize(id, 1);
change the size of the visualizer with the specific id. The size can be (0, 1, or 2)

sg.addToInfo(id, "info");
add information to the pop-up info window

sg.changeInfo(id, "info");
change all content of the information of the pop-up info window

sg.getNumVis();
get the number of visualizers that has been created
